# Todo API (ASP.NET Core + PostgreSQL + Docker)

A production-ready Todo REST API built with ASP.NET Core, PostgreSQL, and Docker Compose.

This project demonstrates backend development, containerization, and deployment-ready architecture using Docker.

---

## 🚀 Features

- ASP.NET Core Web API
- PostgreSQL database
- Entity Framework Core (migrations supported)
- Dockerized backend and database
- Docker Compose orchestration
- Environment-based configuration
- Production-style setup

---

## 🧱 Tech Stack

- ASP.NET Core Web API (.NET)
- PostgreSQL 16
- Entity Framework Core
- Docker
- Docker Compose
